## Full Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Framework | Next.js 15 | Server-side rendering, API routes, app router |
| Language | TypeScript | Type safety across the full stack |
| Styling | Tailwind CSS | Utility-first CSS with CSS variable theming |
| Components | shadcn/ui (new-york) | Accessible, customizable UI primitives |
| AI Model | Claude claude-sonnet-4-20250514 | Tool-use capable LLM for chat |
| AI Protocol | CopilotKit + AG-UI | Agent-UI protocol with observable event streams |
| AI Components | ai-sdk.dev registry | Chat UI elements (messages, prompts, reasoning) |
| Maps | MapLibre GL | Open-source map rendering with dark/light themes |
| Charts | Recharts | Composable chart components for React |
| Tables | TanStack React Table | Headless, performant data tables |
| Backend | Convex | Real-time database, caching, chat persistence |
| Web Scraping | Bright Data MCP | Supplementary data from web sources |
| Data | ArcGIS FeatureServer | 40+ Montgomery city datasets |
| Animations | motion/react | Scroll-driven animations (homepage) |
| Icons | Lucide React | Consistent iconography |
| Deployment | Vercel + Convex Cloud | Frontend hosting + backend infrastructure |

## Key Innovations

- **AG-UI Protocol** — Industry-first implementation for civic applications
- **Dual AI Backend** — Streaming REST + AG-UI agent running simultaneously
- **Client-Side ArcGIS** — Browser-direct queries bypass cloud IP restrictions
- **Portal Isolation** — Each portal has tailored AI, data, and UI
