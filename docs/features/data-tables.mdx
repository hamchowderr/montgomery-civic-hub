## Overview

Each portal includes a data table view built with TanStack React Table, providing sortable and filterable access to the underlying city datasets.

## Features

- **Sortable columns** — Click headers to sort ascending/descending
- **Filterable** — Search and filter within table data
- **Year filtering** — Integrates with the global year range filter
- **Portal-specific columns** — Each portal displays fields relevant to its audience
- **Responsive** — Tables adapt to available panel width

## Data Source

Tables fetch data client-side via the same ArcGIS client used by maps, ensuring consistency between what users see on the map and in the table.
