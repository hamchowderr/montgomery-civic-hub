## Overview

Each portal includes a guided onboarding tour that walks you through its features step by step. Tours use popover highlights to draw attention to key UI elements.

## How to Start

Look for the **"Take a Tour"** button on each portal page. Clicking it starts a sequence of 5-6 guided steps.

## What Each Tour Covers

All four portal tours follow the same structure:

| Step | What You'll See |
|---|---|
| **Welcome** | Overview of what the portal offers |
| **Map View** | How the interactive map works |
| **Table View** | How to browse data in tables |
| **Chart View** | How to read trend visualizations |
| **AI Chat** | How to ask questions and what to expect |
| **Try It** | Example question to get you started |

## Portal-Specific Highlights

<AccordionGroup>
  <Accordion title="Resident Tour">
    Focuses on service requests, safety data, and civic questions. Example: "What is the flood zone for 100 Dexter Ave?"
  </Accordion>
  <Accordion title="Business Tour">
    Highlights permit data, business licenses, and economic intelligence. Example: "What permits are active near downtown Montgomery?"
  </Accordion>
  <Accordion title="City Staff Tour">
    Emphasizes infrastructure tracking, budget dashboards, and project oversight. Covers operational intelligence features.
  </Accordion>
  <Accordion title="Researcher Tour">
    Focuses on longitudinal data, heat maps, and analytical queries. Highlights dataset citation and methodology transparency.
  </Accordion>
</AccordionGroup>

<Tip>
You can restart the tour at any time by clicking the "Take a Tour" button again.
</Tip>
