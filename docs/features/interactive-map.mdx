## Overview

Every portal includes an interactive map powered by MapLibre GL. Data points are plotted directly from Montgomery's ArcGIS datasets, showing real locations and boundaries across the city.

## Map Features

### Color-Coded Data Points

Data points are color-coded by status or category using a consistent semantic color system:

| Color | Meaning |
|---|---|
| **Green** | Active, open, completed |
| **Blue** | In progress, issued |
| **Amber** | Pending, on hold |
| **Red** | Closed, expired, violation |
| **Gray** | Unknown or not applicable |

### Layer Toggling

Each portal has multiple data layers you can show or hide. Use the layer toggle buttons above the map to control which datasets are visible. The AI can also toggle layers for you when answering questions.

### Popups

Click on any data point to see details in a popup. Popup content varies by dataset — you might see addresses, status values, dates, or category information.

### Zoom and Pan

- **Scroll** to zoom in and out
- **Click and drag** to pan around the city
- **Double-click** to zoom in on a specific area
- The map supports **fullscreen mode** for detailed exploration

## Dark Mode

The map automatically syncs with your theme preference. Dark mode uses a dark basemap for comfortable viewing in low-light conditions.

## Council District Overlay

Every portal map includes a toggleable council districts layer showing Montgomery's 9 districts. See the [Council Districts](/features/council-districts) page for details.
