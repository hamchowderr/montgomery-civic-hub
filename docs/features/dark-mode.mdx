## Overview

The Montgomery Civic Hub supports light and dark themes. The entire interface — including the interactive map — adapts to your preference.

## How to Toggle

Use the **theme toggle button** in the top navigation bar to switch between light and dark mode.

## System Preference

The platform respects your operating system's theme preference by default. If your system is set to dark mode, the Civic Hub will start in dark mode.

## Map Adaptation

The MapLibre map automatically switches basemap styles when you toggle themes:
- **Light mode** — Light street map for daytime use
- **Dark mode** — Dark basemap for comfortable viewing in low-light conditions

Data points, district overlays, and all visual layers render correctly in both themes.
