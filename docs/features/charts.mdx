## Overview

Each portal includes chart visualizations built with Recharts, displaying trends, distributions, and comparisons from Montgomery city data.

## Features

- **Multiple chart types** — Bar, line, and area charts per portal
- **Year filtering** — Charts update based on the global year range selector
- **Loading states** — Skeleton bars during data fetch
- **Error handling** — Graceful error states with retry options
- **Portal-specific** — Each portal shows charts relevant to its audience (crime trends, permit volumes, budget breakdowns, demographic distributions)
