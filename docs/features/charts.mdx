## Overview

The Chart tab in each portal provides interactive data visualizations powered by Recharts. Charts show trends, distributions, and patterns across Montgomery's datasets.

## Features

### Interactive Tooltips

Hover over any data point to see exact values in a tooltip. Tooltips show the data label, value, and any relevant context.

### Year Filter Integration

Charts respond to the global [Year Filter](/features/year-filter). Adjust the date range to focus on specific time periods — useful for comparing pre-pandemic vs. post-pandemic trends.

### Portal-Specific Visualizations

Each portal shows charts relevant to its audience:

| Portal | Chart Focus |
|---|---|
| **Resident** | 311 request trends by category and status |
| **Business** | Permit and license activity over time |
| **City Staff** | Infrastructure project timelines, violation trends |
| **Researcher** | Longitudinal patterns across multiple datasets |

## Loading and Error States

Charts show skeleton loading animations while data is being fetched. If a query fails, you'll see an error message with the option to retry.
