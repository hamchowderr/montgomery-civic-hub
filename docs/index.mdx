## Overview

Montgomery Civic Hub is a four-portal civic intelligence dashboard that gives residents, businesses, city staff, and researchers AI-powered access to real-time city data through conversational chat, interactive maps, data tables, and charts.

## Key Features

- **Four specialized portals** — Resident, Business, City Staff, and Researcher views
- **AI-powered chat** — Natural language queries answered with real city data
- **Interactive maps** — MapLibre GL visualizations with council district overlays
- **Live data** — Direct queries to Montgomery's ArcGIS FeatureServer (40+ datasets)
- **Year filtering** — Explore historical trends from 2018 to present
- **Guided tours** — Onboarding walkthroughs for each portal

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 15, TypeScript, Tailwind CSS, shadcn/ui |
| AI | Claude claude-sonnet-4-20250514 with tool use, CopilotKit + AG-UI protocol |
| Data | ArcGIS FeatureServer (40+ datasets), Bright Data web scraping |
| Backend | Convex (real-time sync, caching, chat persistence) |
| Maps | MapLibre GL with dark/light theme sync |
| Deployment | Vercel + Convex Cloud |
